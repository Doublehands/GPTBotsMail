⚡ Deploy to Vercel

📋 部署步骤：

  1. 点击上面的按钮 - 会打开Vercel部署页面
  2. 登录Vercel - 使用GitHub账号登录
  3. 确认设置 - 保持默认设置
  4. 点击Deploy - 等待1-2分钟
  5. 复制URL - 部署成功后复制项目URL

🔧 部署后配置：

将获得的URL更新到您的 api-config.js：

baseUrl: 'https://your-vercel-url.vercel.app/api',

🧪 测试API：

部署完成后，在浏览器控制台运行：

fetch('https://your-vercel-url.vercel.app/api/conversation', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer app-6GQY5ONwN73Spp7Li9Bz8o37'
  },
  body: JSON.stringify({user_id: 'test-user'})
}).then(r => r.json()).then(console.log)
